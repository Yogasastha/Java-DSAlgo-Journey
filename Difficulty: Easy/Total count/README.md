<h2><a href="https://www.geeksforgeeks.org/problems/total-count2415/1?page=2&company=Zoho&sortBy=difficulty">Total count</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">You are given an array<strong> arr[]</strong> of positive integers and a threshold value <strong>k</strong>. For each element in the array, divide it into the <strong>minimum </strong>number of small integers such that each divided integer is less than or equal to <strong>k</strong>. Compute the total number of these integer across all elements of the array.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>

<strong>Output:</strong> 14
<strong>Explanation:</strong> Each number can be expressed as sum of different numbers less than or equal to k as 5 (3 + 2), 8 (3 + 3 + 2), 10 (3 + 3 + 3 + 1), 13 (3 + 3 + 3 + 3 + 1). So, the sum of count of each element is (2+3+4+5)=14.
